# StructureDataExporter

供 **GTNH** 使用的模组，支持导出游戏内大部分方块的烘焙几何数据，用于网页端渲染。



## 游戏内命令 `/sde`


| 子命令 | 说明 |
|--------|------|
| `pos1` / `pos2` | 以**脚下方块**为选区角点。 |
| `hpos1` / `hpos2` | 以**准星指向的方块**为选区角点。 |
| `start` | 开始一次导出会话。 |
| `end` | 结束当前导出会话。 |
| `setname` | 设置输出文件名（无后缀）。 |
| `record` | 将当前选区写入活动帧。 |
| `export` | 写出场景文件。 |
| `web` | 启动本机 Web。 |
| `webstop` | 停止本机 Web 服务。 |
| `sel` | 选区选择模式，Cuboid和Extend模式 |

## 使用说明

- 使用 `start` 开始导出会话
- 使用 `pos1` / `pos2`（站立点）或 `hpos1` / `hpos2`（准星）选定选区
- 选区后必须 `record` 后才能 `export`
- 使用 `web` 打开工作台，选择文件后查看结果
